Pointers & Arithmetic

A pointer is a variable that contains an address of another variable.

Unlike Java, an array variable in C can either store a pointer to the array or the array itself. An array can be accessed in one of two ways:

Using bracket notation:

int * a;
a[4] = 5

Or performing arithmetic on the pointer:

int * a;
*(a + 4) = 5

This suggests that adding a number x to a pointer is the same as adding by x * 4 (for int pointers).

For some variable i and pointer j:

  • &i returns the address of i
  • *j returns the value pointed to by j

Pointer Arithmetic

For int pointers, adding/subtracting an integer to a pointer will increase the value of the pointer by 4 * the integer:

int* a;
a = malloc(10*sizeof(int)); // say at 0x1000
a++; // a == 0x1004 == a[1]
a--; // a == 0x1000 == a[0]

Adding/subtracting two pointers together will yield the difference in index:

int *a, *b;
a = malloc(10*sizeof(int));
a = &a[7];
b = &a[2];
a - b == 5;
