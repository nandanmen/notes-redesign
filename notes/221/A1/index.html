Pointers

Say we declared a normal variable. After its declaration, we can say that the variable is synonymous with its value:

int a = 5
cout << a << endl;
// Outputs 5

We can modify this value:

a = a * 2;
cout << a << endl;
// Outputs 10

The value of this variable lives in a memory address. The & operator can be used to access it:

cout << &a << endl;

What is a pointer?

A pointer is a variable that holds the address of another variable. It is declared using the * operator before an identifier:

int* aPointer;
cout << aPointer << endl;
// Outputs 0x0, as the pointer currently points nowhere

We can set the address the pointer points to using the & operator:

int* aPointer = &a;
cout << aPointer << endl;
// Outputs the address of a

Another way we can declare a pointer is using the new operator:

int* newPointer = new int(3);

Modifying pointer values

We can modify the value the pointer points to using the dereference operator:

*aPointer = 2;
// a = 2 now

This code is essentially saying "change the value at the address held by aPointer to 2". So essentially the same as a = 2.

Why use pointers?

In C++, when variables are passed into a function, a copy of the variable is passed. The original variable is untouched.

void setValueToFive(int x) {
  x = 5;
}

int x = 3;
setValueToFive(x);

cout << x << endl;
// Outputs 3, x was unchanged

The function can be modified to accept a pointer instead of the actual variable. This results in the actual value being changed:

void setValueToFive(int* x) {
  *x = 5;
}

int x = 3;
setValueToFive(x);

cout << x << endl;
// Outputs 5, x was changed

Using references

Another way to change our function is to instead use a reference to our variable like so:

void setValueToFive(int& x) {
  x = 5;
}
        
int x = 3;
setValueToFive(x);
        
cout << x << endl;
// Outputs 5, x was changed

A reference is essentially adding another name to our variable. Changing this other name changes the original variable and vice versa:

int a = 4;
int& b = a;

b = 5;
cout << a << endl; // outputs 5
