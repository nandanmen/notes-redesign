Abstract Data Types

Beginner Algorithms and Data Structures
221.7

Idea

An abstract data type is a collection of data and a set of operations on that data — it describes what data gets collected and what operations to perform, but not how the data is stored or how the operations are performed.

Operations

Mutators

Also known as setters, it changes the contents of an ADT generally by adding or removing data in the collection.

Accessors

Also known as getters, it is able to retrieve data from the ADT and ask questions about the data collection (without manipulating the contents itself).
