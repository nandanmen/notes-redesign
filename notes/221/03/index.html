Function Calling

Function Parameters

C++ differentiates between actual parameters and formal parameters.

  • Actual parameter — the value or variable(s) that are specified by the function caller.
  • Formal parameter — the variables found in the signature/header of the function itself.

Formal parameters must match with actual parameters in order, number, and data type. To make it more clear, consider the following code snippet:

int a = 5;
int b = 7;
int result = sum(a, b);

Here, a and b are considered to be actual parameters. Below, x and y are considered to be formal parameters:

int sum(int x, int y) {
  return x + y;
}

Actual and formal parameters differ in memory, even if they are named the same. The value of the actual parameter is copied to the formal parameter when the function is called.

C++ Arrays

An array is a collection of elements of the same data type, stored in consecutive locations in memory. They are declared like regular variables but with [], which includes the size of the array:

int age[100]
