Proving Big– Theorems

Generally proving big– theorems require you to simply use their respective definitions. This requires choosing values of n0 and c to satisfy the inequality. Sometimes these values are not so obvious, so there are several tricks to employ to find them.

Proving Big-Oh

For proving functions that are polynomials, simply pick n0 as 1 and c as the sum of the coefficients.

For example, say we want to prove the following theorem:

n3 + 20n + 1 ∈ O(n3)

Let n0 = 1, and c = 1 + 20 + 1 = 22. Now we can see that the inequality is satisfied:

n3 + 20n + 1 ⋜ n3 + 20n3 + n3 = 22n3

We know this is satisfied because for n > 1, then:

n3 > n² > n > 1

Using Limits

Another way to approach Big– proofs is to solve for c then taking the limit to infinity.

For example, let's say we want to prove the following:

f(n) = 3log2n ∈ Ω(g(n) = 3(n+1))

Essentially we need to find a c and n0 such that:

For all n >= n0, 3log2n > 3c(n+1)

Note that we can rewrite f(n) in an easier form:

3log2n = n1/log32 > n for n > 1

Since we know this, we just need to compare n to g(n):

n >= 3c(n+1)

Now if we solve for c, then we find that:

n >= 3nc + 3c
n - 3nc >= 3c
n(1 - 3c) >= 3c
n >= 3c / 1 - 3c

Now note that if we take the limit for n to infinity, then c will tend towards 1/3. Therefore the above inequality will only hold as long as c < 1/3.

Now that we have a range of c to choose from, we can just pick any arbitrary c that is in this range.

Simple Example

Using a simpler example we can see how useful the trick to use limits is. Let's say we want to prove the following by contradiction:

3n2 + 2n ∉ O(n)

Let's assume the opposite is true. By the definition of Big O:

3n2 + 2n <= cn

Now solving for c and taking the limit as n tends to infinity we find that:

limn→∞(3n + 2) = ∞

Therefore for any value c we can find a value 3n+2 larger than c. This is a contradiction, and therefore:

3n2 + 2n ∉ O(n)
