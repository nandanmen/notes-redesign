Proving Big– Theorems

Generally proving big– theorems require you to simply use their respective definitions. This requires choosing values of n0 and c to satisfy the inequality. Sometimes these values are not so obvious, so there are several tricks to employ to find them.

Proving Big-Oh

For proving functions that are polynomials, simply pick n0 as 1 and c as the sum of the coefficients.

For example, say we want to prove the following theorem:

n3 + 20n + 1 ∈ O(n3)

Let n0 = 1, and c = 1 + 20 + 1 = 22. Now we can see that the inequality is satisfied:

n3 + 20n + 1 ⋜ n3 + 20n3 + n3 = 22n3

We know this is satisfied because for n > 1, then:

n3 > n² > n > 1
